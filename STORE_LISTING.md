## Short Description (132 chars max)

Full-featured color picker with eye dropper, multiple formats (HEX, RGB, HSL, Tailwind), contrast checker, color harmonies, palettes & export.

---

## Detailed Description

```
Color Picker — the essential tool for designers and developers.

Pick any color from your screen with a single click and copy it in the format you need. No account required, works completely offline.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⎯ EYE DROPPER TOOL
Click anywhere on any webpage to capture the exact color. Works on images, gradients, videos, and any visible element.

⎯ MULTIPLE COLOR FORMATS
• HEX (#667EEA)
• RGB / RGBA
• HSL / HSLA
• HEX8 (with alpha)
• Tailwind CSS classes
• CSS custom properties

⎯ TAILWIND CSS INTEGRATION
Automatically finds the closest Tailwind color match. Perfect for Tailwind CSS projects.

⎯ CONTRAST CHECKER
Check WCAG accessibility compliance instantly:
• AA compliance (4.5:1 ratio)
• AAA compliance (7:1 ratio)
• Real-time contrast ratios against white and black

⎯ COLOR HARMONIES
Generate harmonious color combinations:
• Complementary
• Analogous
• Triadic

⎯ COLOR ADJUSTMENTS
Fine-tune colors with lightness and saturation sliders. Preview changes before applying.

⎯ COLOR HISTORY
Automatically saves your recently picked colors. Click any color to copy it instantly.

⎯ CUSTOM PALETTES
Organize colors into named palettes. Perfect for managing brand colors or project themes.

⎯ EXPORT OPTIONS
Export your colors to:
• JSON
• CSS Variables
• SCSS Variables
• Tailwind Config

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

KEYBOARD SHORTCUTS

• Ctrl+Shift+C (Cmd+Shift+C on Mac) — Open Color Picker
• P — Pick a color
• C — Copy current color

✓ No data collection
✓ No account required
✓ No external servers

Perfect for:
```
